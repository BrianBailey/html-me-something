Matplotlib

The more you get into manipulating data the more you want to actually be able to visualize all of this data. The library matplotlib is excellent for this. It also is built into the SciPy stack and you can see some incredible data plotted out that is worth "thousands of words and comments".

I recently started going through some tutorials. One of the first ones is going to be the iris data set. Using a library like numpy, you are able to organize and visualize all the data in this csv file. I want to continue practicing machine learning. It is a lot of work to start to understand the algorithms that are used but I believe this is all definitely the future of harnessing the power of computers.
